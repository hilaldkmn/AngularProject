Angular kurulumu

Node.js indirilme ile başlanır. Şu adrese girilir. — https://nodejs.org/en/download/ . Ve kurulur.

Sonra terminal ekranından “npm install -g typescript” typescript kurulur.

Gerekli hazırlıklar yapıldıktan sonra sıra ile terminal ekranına Angular'ı kurmaya yarayacak komutlar:

npm install -g @angular/cli

ng new Proje_ismi

cd Proje_ismi

ng serve

Kurulum ve çalıştırma başarı ile tamamlanmıştır.
